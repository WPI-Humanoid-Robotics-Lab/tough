Detailed Description

The LightDetect class detects lights seen on a lightpanel. The rgb value of the detected light is published on topic "/detect/light/rgb" and its position is published on topic "/detect/light/xy". The node subscribes to "/multisense/camera/left/image_raw" topic for images.
